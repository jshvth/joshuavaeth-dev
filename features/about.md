# About

## Summary
The about section provides a concise profile with a two-column text layout and a portrait image treatment that shifts from grayscale to color on hover.

## Highlights
- Profile headline and short bio split into two columns on larger screens.
- Portrait card with subtle border, overlay gradient, and hover reveal.
- Section divider label for clear navigation context.

## Implementation Notes
- Component: `src/components/About.jsx`
- Portrait: `public/joshua.jpg` with fallback placeholder

## Behavior
- Responsive layout that stacks on small screens.
- Image desaturates by default and restores color on hover.
