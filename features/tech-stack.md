# Tech Stack

## Summary
The tech stack section groups skills into frontend, backend, and tools with icons and concise labels.

## Highlights
- Clear category grouping with uppercase headers.
- Icon-driven grid for fast visual scanning.
- Hover scale for a subtle interactive feel.

## Implementation Notes
- Component: `src/components/Stack.jsx`
- Icons: `react-icons` with `Si*` and `Tb*` sets

## Behavior
- Responsive grid that expands with screen size.
- Categories render from a single data array.
