# Experience

## Summary
The experience section presents a vertical timeline-like list of roles with emphasis on the current role.

## Highlights
- Large typography and strong visual hierarchy for roles and periods.
- Subtle hover state on each entry.
- Animated status indicator for the current role.

## Implementation Notes
- Component: `src/components/Experience.jsx`
- Data-driven layout using a local `experiences` array

## Behavior
- Entries stack on small screens and spread into columns on larger screens.
- Current role is highlighted via color and ping animation.
