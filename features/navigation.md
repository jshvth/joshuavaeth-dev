# Navigation

## Summary
The navigation system uses a fixed top bar with a full-screen overlay menu for focused browsing.

## Highlights
- Minimal logo mark linking back to the hero section.
- Fullscreen menu with large typography and clean spacing.
- Includes a prominent external support link.

## Implementation Notes
- Component: `src/components/Navbar.jsx`
- Hamburger menu uses `react-icons` for open/close states

## Behavior
- Overlay slides in and locks focus on navigation items.
- Clicking any menu item closes the overlay.
